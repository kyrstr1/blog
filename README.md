# 🌐 Minimal Blog Web Site

Modern, hızlı ve responsive bir blog web sitesi.  
Temiz tasarım + akıcı kullanıcı deneyimi odaklı geliştirilmiştir.

---

- ⚡ Hızlı ve hafif yapı

- 🎨 Minimal ve modern UI
- 🌌 Animasyonlu arka plan
- 🔗 Özel tasarlanmış butonlar
- 🧩 Kolay düzenlenebilir yapı

---

## 🛠️ Kullanılan Teknolojiler

- HTML5
- CSS3 (Animasyonlar & Responsive)
